# SMS_project

Project "Hệ thống hỗ trợ giáo dục" gồm 3 chức năng chính:
- Hệ thống quản lí dữ liệu sinh viên
- Diễn đàn
- Trợ lý bằng giọng nói

# Chức năng quản lí dữ liệu sinh viên
Các tính năng:
- Thêm / xóa sinh viên khỏi hệ thống
- Tìm kiếm thông tin
- Cập nhật thông tin
- Xuất file dữ liệu
